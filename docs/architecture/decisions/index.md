# Architecture Decision Records

Thư mục này chứa các Architecture Decision Records (ADRs) — ghi lại các quyết định kiến trúc quan trọng trong gitops repo.

## Format

Mỗi ADR là một file markdown `XXXX-<tên-quyết-định>.md`:

```markdown
# ADR-XXXX: <Tiêu đề>

## Trạng thái

Accepted | Superseded | Deprecated

## Bối cảnh

Vấn đề cần giải quyết.

## Quyết định

Chọn phương án nào và tại sao.

## Hệ quả

Tác động tích cực và tiêu cực.
```
